Fast Order

The web-application Fast Order is an application for barkeepers as well as bar visitors.

It can be used by a visitor to order drinks and by an admin to manage the orders.

This app was developed with a total of 5 people in 6 months, as a group project in university.

I personally focused on the barkeeper's view part, but we all worked on the project together, which taught me how to use git and solve merge

conflicts, as well as teamwork.

Features include: Live synchronization with other workers, automatically updated stock of drinks & easy to use and understand UI/UX

For more information or to try it yourself, please visit the projects github page: https://github.com/mantheij/fastorder
